KRETA KIT

Kreta Kit es una herramienta diseñada para optimizar y automatizar el flujo de carga y gestión de productos en tiendas e-commerce, enfocada en dueños de negocios que necesitan escalar su catálogo de forma rápida y eficiente. 06 / SAAS & COMMUNICATIONS

WACENTER

Solución técnica unificada para la gestión de comunicaciones WhatsApp, diseñada para empresas que requieren interacción bidireccional escalable. Emplea la API de WhatsApp Cloud con persistencia en MySQL, ofreciendo un panel estilo inbox desarrollado en Next.js con soporte para mensajes en vivo vía Socket.IO. El backend en PHP se integra con n8n para automatización inteligente, permitiendo respuestas automáticas basadas en datos o intervención humana en tiempo real.

SOBRE MÍ

Construyo productos digitales donde el diseño y la ingeniería comparten el mismo nivel de exigencia.
